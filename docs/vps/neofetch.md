# Как устоновить NeoFetch?

## Что такое neofetch?

- **Neofetch** — это инструмент командной строки для отображения системной информации в терминале.

## Через apt

1. Войдите в вашу VDS
2. Введите:
```sh
apt install neofetch -y
```
3. Нажмите Enter
4. Ожидайте устоновки
5. Пропишите: `neofetch`
6. Готово!
